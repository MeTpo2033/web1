<strong>과 같은 태그는 처음 열리는 태그 뒤에 닫히는 태그는 /를 붙힘 </strong>
태그는 태그안에 있는 문장을 설명함. 태그의 이름인 이미 설명하는 이름!
태그나 컴퓨터 공학에서의 프로그래밍 언어의 명령어들은 하나의 시의 언어임!

h1 ~ h6 == 제목, 숫자가 작을수록 작아짐, h태그는 제목을 나타냄(진하게 표현)
strong 태그는 글씨 강조
u 태그는 밑줄
br 줄바꿈 , 닫히는 태그는 불필요 -> 그저 줄바꿈(많이 할수있음)
p 태그는 단락을 표현, 열리는태그, 닫히는태그 존재 -> p태그는 단락을 표현하기에 HTML을 좀더 정보있게 표현가능(여백이 정해져있음 -> 
CSS활용해야됨) (margin-top=40px 은 여백을 위쪽에다 40만큼)

HTML의  태그를 의미있게 사용해야 비즈니스측면에서의 접근성, 휴미니즘측면에서의 접근성을 모두 만족시킬수있음!

태그가 태그의 이름만으로 정보가 부족할때 태그옆에 속성을 넣어 정보를 추가할수 있음
ex. <img src = "coding.jpg" width = "100%" > 속성의 위치는 상관없고 width는 사진의 크기를 정해주고 <img>태그는 이미지를 삽입할때 사용

<ul> 태그는 부모태그, 자식태그인 <li> 태그를 가짐 <li>태그는 목록을 만들기 위함, <ul>태그는 구분지어줌
<ol>태그는 안에 있는 목록을 시작할때 1. 로 바꿔줌 
<ul> 은 unordered lit <ol>은 ordered list !!

<table>
	<tr>
		<td>
관계의 부모,자식 태그관계 - 테이블은 표를 작성할때 쓰임 tr은 줄바꿈, td는 한줄에서의 영역나누기, table은 한개의 표

<title> 태그는 웹페이지의 제목
<meta> 태그는 웹사이트의 읽는 방식을 바꿔줌 ( <meta charset="utf-8"> -> utf-8로 문서를 읽어라 (char = 문자, set= 정하다)

<head> 태그는 title, meta등의 본문을 설명하는 코드를 감싸며
<body> 태그는 본문의 내용과 그 태그들을 감쌈
이 모든걸 감싸는 태그는 <html>
<!doctype html>을 맨위에 써서 이 문서는 html이라는것을 암시함

링크는 <a> 태그, href 속성을 사용하여 링크를 걸수 있음, 새탭은 target = "_blank" 속성, 하이퍼링크 툴팁은 title 속성
